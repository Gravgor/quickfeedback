Note: The feedback widget should display "Powered by Acme Inc" instead of the default QuickFeedback branding.

Custom Company Branding

The QuickFeedback widget allows you to customize the branding that appears in the widget. This is done by setting the data-company attribute in the script tag.

<script src="http://localhost:3000/widget.js" data-site-id="e6041cff-c3f1-49a1-82b2-eed2d74ebdac" data-position="bottom-left" data-color="#10b981" data-company="Acme Inc"> </script>
