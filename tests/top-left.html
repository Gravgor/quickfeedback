QuickFeedback Widget Test - Top Left Position

This page demonstrates the widget in the top-left position with a different color.

Note: The feedback button should appear in the top-left corner of the screen with a purple color.

Widget Positioning

The QuickFeedback widget can be positioned in four different locations:

  • bottom-right (default)
  • bottom-left
  • top-right
  • top-left (shown on this page)

Widget Color Customization

The widget color can be customized using the data-color attribute. On this page, we've set it to #8b5cf6 (purple).
